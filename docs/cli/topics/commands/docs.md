# Docs command

Use `wavepeek docs` when command help is not enough.

## Browse and search

Start with `wavepeek docs topics` to list available topics or `wavepeek docs
search <query>` when you only remember part of the topic name or heading.

## Export when you need files

Use `wavepeek docs export <OUT_DIR>` only when you need the authored Markdown on
disk. Routine reading and discovery should stay inside the installed binary.
