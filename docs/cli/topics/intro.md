# Introduction

`wavepeek` exposes documentation in three layers.

## Start with help

Use `wavepeek -h` when you need a compact syntax reminder. Use `wavepeek --help`
or `wavepeek help <command>` when you need the fuller command reference.

## Use local docs for narrative guidance

Use `wavepeek docs` when you need more than syntax. The embedded topics explain
concepts, workflows, and troubleshooting steps that stay version-matched to the
installed binary.
